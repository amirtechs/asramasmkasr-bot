# asramasmkasr-bot

**What?**

E-Checking Hostel System is an online application by Telegram Application that allow users to check their exam results and performance quickly. This application require user to insert their NRIC in order to check their dormitory status. By using this application, they can access the application anywhere and anytime as long as they connect to the Internet.

**Features**

1. Check the dormitory 
2. Check the warden duty roaster
